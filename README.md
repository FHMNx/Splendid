Splendid is a full-stack money management ( Expense Tracker ) web application designed for small and medium-sized businesses. 
It helps business owners track income, expenses, and financial performance in a simple and organized way.

The system is built using modern industry technologies with a scalable architecture, making it suitable 
for real-world business use and future SaaS deployment.

Key Features:
- Income and expense tracking
- Financial summaries and reports
- Secure user authentication
- Multi-user support
- Clean and responsive dashboard
- Cloud-ready architecture

Tech Stack:
Frontend:
- React.js
- Vite
- Tailwind CSS

Backend:
- Spring Boot
- Hibernate (JPA)

Database:
- MySQL (development)
- PostgreSQL (production)

Deployment:
- Netlify (frontend)
- Render / Cloud platform (backend)

Purpose:
This project is developed as an industry-level full-stack application to demonstrate real-world 
software engineering practices and to be used as a SaaS solution for local businesses.
